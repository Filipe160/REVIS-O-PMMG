O que é um substantivo?

É uma palavra que nomeia seres, objetos, sentimentos, lugares etc. Exemplo: casa, alegria, Brasil.

O que é um verbo?

Indica ação, estado ou fenômeno da natureza. Exemplo: correr, ser, chover.

O que é um adjetivo?

Palavra que caracteriza o substantivo, atribuindo-lhe uma qualidade. Exemplo: bonito, rápido, feliz.

O que é semântica?

É a parte da gramática que estuda o significado das palavras e das frases.

O que é sinonímia?

É a relação de semelhança de sentido entre palavras diferentes. Exemplo: “feliz” e “contente”.

O que é antonímia?

É a relação de oposição de sentido entre palavras. Exemplo: “alto” e “baixo”.

Qual é a função da metáfora?

Criar uma comparação implícita entre dois elementos para deixar a linguagem mais expressiva. Exemplo: “O tempo é um rio que corre sem parar.”

Qual é a função da comparação?

Estabelecer uma relação de semelhança entre dois elementos, usando palavras como “como”, “igual a” ou “tal qual”. Exemplo: “Ela é linda como uma flor.”

Qual é a função da personificação?

Atribuir características humanas a seres inanimados ou animais, tornando o texto mais poético. Exemplo: “O vento cantava entre as árvores.
