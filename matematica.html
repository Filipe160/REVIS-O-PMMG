Quanto é 20% de 150?

20% de 150 é 30.

O que é uma proporção?

É uma igualdade entre duas razões. Exemplo: 2/4 = 3/6.

Conjunto e Estatística

Clique no botão para ver a resposta de cada pergunta.

O que é um conjunto na matemática?

É uma coleção de elementos com uma característica em comum. Exemplo: o conjunto dos números pares {2, 4, 6, 8...}.

O que é a média aritmética simples?

É a soma de todos os valores dividida pela quantidade deles. Exemplo: (4 + 6 + 8) ÷ 3 = 6.

O que é o conjunto universo?

É o conjunto que contém todos os elementos possíveis em uma situação. Exemplo: se o tema é “números naturais”, o conjunto universo é {0, 1, 2, 3...}.
